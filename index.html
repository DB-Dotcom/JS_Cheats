JavaScript frequently used Methods Cheat Sheets

JavaScript String Methods

Method Description Example Output
split() Splits a string into an array of substrings based on a specified separator. "Hello World".split(" ") [ "Hello", "World" ]
concat() Combines two or more strings and returns a new string. "Hello".concat(" ", "World") "Hello World"
toUpperCase() Converts a string to uppercase. "Hello".toUpperCase() "HELLO"
toLowerCase() Converts a string to lowercase. "Hello".toLowerCase() "hello"
indexOf() Returns the index of the first occurrence of a specified value in a string. "Hello World".indexOf("World") 6
substring() Returns a substring of a string between two indices. "Hello World".substring(0, 5) "Hello"
replace() Replaces a specified value or regular expression with a replacement in a string. "Hello World".replace("World", "Universe") "Hello Universe"
trim() Removes whitespace from both ends of a string. " Hello ".trim() "Hello"
charAt() Returns the character at a specified index in a string. "Hello".charAt(1) "e"
slice() Returns a portion of a string between two indices. "Hello World".slice(6) "World"

JavaScript Array Methods

Method Description Example Output
push() Adds one or more elements to the end of an array and returns the new length. arr.push(5) Array length increased by 1
pop() Removes the last element from an array and returns that element. arr.pop() Last element of the array
shift() Removes the first element from an array and shifts the remaining elements to a lower index. arr.shift() First element of the array
unshift() Adds one or more elements to the beginning of an array and returns the new length. arr.unshift(3) Array length increased by 1
concat() Combines two or more arrays and returns a new array. arr.concat([6, 7]) New combined array
join() Joins all elements of an array into a string using a specified separator. arr.join(", ") Joined string
map() Creates a new array by applying a function to each element of an existing array. arr.map(x => x * 2) New array with mapped values
filter() Creates a new array with all elements that pass a test implemented by a provided function. arr.filter(x => x > 3) New array with filtered values
reduce() Applies a function to an accumulator and each element in the array to reduce it to a single value. arr.reduce((acc, val) => acc + val, 0) Reduced value
forEach() Executes a provided function once for each array element. arr.forEach(item => console.log(item)) No specific output, side effect

JavaScript Object Methods

Method Description Example Output
Object.keys() Returns an array of keys from an object. Object.keys({a: 1, b: 2}) [ "a", "b" ]
Object.values() Returns an array of values from an object. Object.values({a: 1, b: 2}) [ 1, 2 ]
Object.entries() Returns an array of key-value pairs from an object. Object.entries({a: 1, b: 2}) [ [ "a", 1 ], [ "b", 2 ] ]
Object.assign() Copies the values of all enumerable properties from one or more source objects to a target object. Object.assign({}, {a: 1, b: 2}) { a: 1, b: 2 }
hasOwnProperty() Returns a boolean indicating whether an object has a specific property as a direct property. obj.hasOwnProperty("a") true or false
Object.freeze() Freezes an object, preventing new properties from being added and existing properties from being modified or deleted. Object.freeze({a: 1, b: 2}) Immutable object
Object.seal() Seals an object, preventing new properties from being added and existing properties from being deleted. Object.seal({a: 1, b: 2}) Object is sealed
Object.create() Creates a new object with the specified prototype object and properties. Object.create(proto, {prop: {value: 42}}) New object with prototype and properties

JavaScript Function and Callback Methods

Method Description Example Output
setTimeout() Executes a function after a specified time delay. setTimeout(() => console.log("Delayed message"), 1000) Message after 1 second
setInterval() Repeats a function at specified intervals. setInterval(() => console.log("Repeated message"), 2000) Message every 2 seconds
clearTimeout() Clears a timeout previously set by setTimeout(). clearTimeout(timeoutID) Timeout cleared
clearInterval() Clears an interval previously set by setInterval(). clearInterval(intervalID) Interval cleared
addEventListener() Attaches an event handler to an element. element.addEventListener("click", () => console.log("Clicked")) Message on click
removeEventListener() Removes an event handler from an element. element.removeEventListener("click", handler) Event handler removed
Promise.resolve() Returns a resolved Promise with a value. Promise.resolve("Resolved value") Resolved Promise with value
Promise.reject() Returns a rejected Promise with a reason. Promise.reject("Reason for rejection") Rejected Promise with reason
fetch() Initiates a network request and returns a Promise that resolves to the response. fetch("https://api.example.com/data") Promise with fetched data

JavaScript DOM Manipulation Methods

Method Description Example Output
getElementById() Returns a reference to the element by its ID. document.getElementById("myElement") Element with ID "myElement"
querySelector() Returns the first element within the document that matches the specified selector. document.querySelector(".myClass") Element with class "myClass"
querySelectorAll() Returns a NodeList containing all elements that match the specified selector. document.querySelectorAll("p") All

elements
createElement() Creates a new element with the specified tag name. document.createElement("div") New ("div") element
appendChild() Adds a new child node to an element. parentElement.appendChild(newChild) Child appended to parent
removeChild() Removes a child node from an element. parentElement.removeChild(child) Child removed from parent
classList.add() Adds one or more classes to an element's class attribute. element.classList.add("active") Class "active" added
classList.remove() Removes one or more classes from an element's class attribute. element.classList.remove("inactive") Class "inactive" removed
setAttribute() Sets the value of an attribute on an element. element.setAttribute("data-id", "123") Attribute set
removeAttribute() Removes an attribute from an element. element.removeAttribute("data-id") Attribute removed

Array-Destrukturierung und Spread-Operator

Method Description Example Output
...array Spread operator expands an array into individual elements. const newArray = [...oldArray, 4, 5] [1, 2, 3, 4, 5]
const [item1, item2] = array Array destructuring assigns array elements to variables. const [first, second] = [1, 2] first = 1, second = 2

Objekt-Destrukturierung

Method Description Example Output
const { prop1, prop2 } = object Object destructuring assigns object properties to variables. const { name, age } = { name: "John", age: 30 } name = "John", age = 30

Promises und async/await-Methoden

Method Description Example Output
then() Attaches a callback for the resolution of a Promise. promise.then(result => console.log(result)) Result of resolved Promise
catch() Attaches a callback for the rejection of a Promise. promise.catch(error => console.error(error)) Error from rejected Promise
finally() Attaches a callback that is invoked when a Promise is settled. promise.finally(() => console.log("Done")) Message after Promise is settled
async function Defines an asynchronous function that returns a Promise. async function fetchData() { /* ... */ } Asynchronous function
await Pauses the execution of an async function until the Promise is resolved. const result = await fetchData() Resolved value from Promise

Reguläre Ausdrücke (Regex)

Method Description Example Output
test() Tests if a string matches a regular expression. /abc/.test("abcdef") true
exec() Executes a search for a match in a string. /abc/.exec("abcdef") ["abc"]
match() Searches a string for a match against a regular expression. "abcdef".match(/abc/) ["abc"]
replace() Replaces matches in a string with a new substring. "abcdef".replace(/abc/, "xyz") "xyzdef"
search() Searches a string for a match against a regular expression and returns the index of the match. "abcdef".search(/abc/) 0

Array-Methoden für Funktionale Programmierung

Method Description Example Output
map() Creates a new array with the results of calling a provided function on every element. [1, 2, 3].map(x => x * 2) [2, 4, 6]
filter() Creates a new array with all elements that pass the test of a provided function. [1, 2, 3, 4, 5].filter(x => x % 2 === 0) [2, 4]

forEach() Calls a provided function once for each element in the array. [1, 2, 3].forEach(x => console.log(x)) Logs each element
every() Tests whether all elements in the array pass the test of a provided function. [2, 4, 6].every(x => x % 2 === 0) true
some() Tests whether at least one element in the array passes the test of a provided function. [1, 3, 5].some(x => x % 2 === 0) false
